# {EPITECH} - my_ls

You must recode the ls [-alRdrt] [FILE] command.

I corrected the link to the file address.

The program supports the flags below

If you are a Tek student dont CTRL C + CTRL V

```
./my_ls
./my_ls -a

./my_ls Test
./my_ls -a Test
```




## Installation

Program compile with Makefile, use this command

```bash
  make
  make clean

  ./my_ls
```
